What is Canvas API?

The Canvas API is used to draw 2D shapes right on your webpage! There are only two primitive shapes supported by Canvas, namely rectangles and paths. That may not sound like much, but here's the important point: Virtually every other complex shape can be drawn with paths!

As an aside, you can use the WebGL API to draw 3D shapes!

What can I use Canvas API for?

Canvas API has many use cases!

  1. Draw a box
  2. Draw the Earth
  3. Recreate Mona Lisa

The possibilities are endless!

How to use Canvas API

You may now be wondering how to use this amazing technology. Here is a code snippet to demonstrate how we created the square above: 

          const canvas = document.getElementById("canvas");
          const context = canvas.getContext("2d");

          context.strokeStyle = "red";
          context.lineWidth = 10;
          context.strokeRect(100, 25, 100, 100);

Note that we need to have a corresponding canvas tag in our HTML, otherwise this will not work:

<canvas id="canvas">
